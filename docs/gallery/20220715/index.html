これからのリアルライブ活動について！

Homegallery これからのリアルライブ活動について！

ニューロナがバーチャルの世界で活動を始め、リアルの世界に初めて出てきて約一年。
まだまだ新猫のライブに…足を運んでくれた方、配信で見守ってくれた方、イベントにお誘いいただいたみなさま、本当にありがとうございます。

それまでリアルのライブ経験がなかったニューロナ。まだやり残していることややりたいこともあるのですが、まずは当音楽隊のメンバー・黒猫の黒糖くん（6歳・男の子）の慢性腎不全の治療のため、しばしライブ活動をお休みします。
一番先の目標として、黒糖くんの回復を目指します。

どのくらいの時間がかかるかわからないですが、お昼の時間帯等に活動ができることもあるかと思いますので、タイミングを見て、またみんなに会いに行きたいと思っています。
もちろん、ネットでの活動はそのまま続きます！

グッズ販売を含めライブでいただきました収益は、黒糖くんの療法食・サプリメントおよび投薬の際に必要となるおやつ代に充てさせていただきます。使い道に関しましては今後こちらのウェブサイトで公表させていただきたいと思います。
ありがとう。これからもよろしくお願いします。
